## GPIO初始化
GPIO初始化的大致思想分为如下两步：
1.开启总线时钟
2.配置工作模式

### 基于寄存器方式
不论何种初始化方式，本质上都是对GPIO相关寄存器的配置，因此我们从最原始的方式开始：

```
void GPIO_Init_Config()
{
  RCC->APB2ENR |= RCC_APB2ENR_IOPAEN;
  
  GPIOA->CRL &= 0x00000000;
  GPIOA->CRH |= 0xFFFFFFFF;
  
  GPIOA->CRL |= 0x76543210;
  GPIOA->CRH &= 0xFEDCBA98;
}
```
